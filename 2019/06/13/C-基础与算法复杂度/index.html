考研复习所做笔记，主要参考了王道、天勤和数据结构课本；C++参考了C++ Primer和Effective C++。

C++基础

输入和输出

C++语言并没有定义IO语句，而是使用了一个全面的标准库（iostream）来提供IO机制

  • 标准输入–cin，用作输入信息

  • 标准输出–cout，用作输出信息

  • 标准错误–cerr，输出警告和错误信息

  • 一般输出–clog，输出程序运行的一般信息

    注：1.Windows中输入文件结束符为Ctrl+Z，然后按Enter，UNIX和Mac OS中为Ctrl+D

循环

while

先判断，再执行

do…while

先执行一次，再判断

for

先判断，再执行

循环控制语句

break

跳出当前循环

continue

会跳过当前循环并开始下一次循环。

对于for循环，continue语句会导致执行条件测试和循环增量部分。

对于while和 do…while 循环，continue语句会导致程序控制回到条件测试上。

goto

语句允许把控制无条件转移到同一函数内的被标记的语句。

注意：不建议使用goto语句
