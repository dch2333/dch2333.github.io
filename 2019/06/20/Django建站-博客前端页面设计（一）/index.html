Django建站笔记
  • CSS
  • Django
  • Html
  • JS
  • Python

包括基础页面、主页和404页面。

页面设计

  • 主页：包含头像、项目目录链接、标签目录链接、自我介绍链接和五篇文章。参数：页号
  • 404页：404。参数：无
  • 项目目录页：包含所有项目链接。参数：无
  • 项目页：包含该项目简介及该项目下的所有文章链接。参数：项目名
  • 标签目录页：包含所有标签链接。参数：无
  • 标签页：包含该标签简介及该标签下的所有文章链接。参数：标签名
  • 文章页：包含文章内容。参数：文章名
  • 自我介绍页：包含自我介绍。参数：无
