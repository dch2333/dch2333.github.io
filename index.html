MarkDown语法

一.标题

用#号，#越多标题级别越低，上面“一.标题”我就用的是一级标题
例如：

# 123

## 123

### 123

#### 123

##### 123

###### 123

输出：

123

123

123

123

123
123

二.引用

当你需要引用别人说的话的时候，一般要用这种格式
在’>’后加一个空格再加你要引用的话
例如：
> 就像这样

就像这样

三.列表

文字前加 * 号，例如：

* 1

* 2

* 3

  • 1
  • 2
  • 3

四.表格

`

| Item | Value | Qty |

| :——– | ——–:| :–: |

| Computer | 1600 USD | 5 |

| Phone | 12 USD | 12 |

| Pipe | 1 USD | 234 |

`

Item Value Qty
Computer 1600 USD 5
Phone 12 USD 12
Pipe 1 USD 234

五.代码块

用三个 ` 围起来或者直接缩进4个空格

例：

```
include<iostream>

using namespace std;
```

七.图片&链接

插入图片：

![]()

不举例了，在小括号里放图片链接就行==

然后是插入链接(以百度为例)：

[百度一下](https://www.baidu.com/)

输出就是：百度一下

八.粗体&斜体

两个 * 包含一段文本是粗体，用一个 * 包含一段文本是斜体

例如：

**粗体**

输出：

粗体

例如：

*斜体*

输出：

斜体

九.复选框

使用 - [ ] 和 - [x] 语法可以创建复选框，实现 todo-list 等功能。例如：

  • 已完成事项
  • 待办事项1
  • 待办事项2

十.使用心得

1.各级标题和代码块之间一定要空行！引用和代码块也要空行！总之就是各种空行！！！

2.代码块后面要空两行才能用标题，不然标题就会被放到代码块里，不知道为啥？—？
